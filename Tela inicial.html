Seja bem vindo ao Caixa Multibanco Eletrônico

Para começar faça login no seu banco

Acesse atraves do Postman ou qualquer outro API Client

Login: Com o metodo POST

/banco/loginBanco

Exemplo de requisição:

{
  "login": "exemplo_login",
  "senha": "S3nh@.d0-Cl13nt3",
  "caixa": 1,
  "conta": "Conta do Cliente"
}

Consultar Saldo: Com o metodo GET

/banco/consultarSaldo

Exemplo de requisição:

{
  "acesso": "EDDA8F2DA1428C789F3D599432188D1B687BCCD18ECD02400498721AB4C60D59"
}
